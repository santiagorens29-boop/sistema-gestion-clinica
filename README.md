# 🏥 Sistema de Gestión Clínica

Este es un sistema integral desarrollado para la administración eficiente de centros médicos, permitiendo la gestión de pacientes, turnos e historias clínicas de forma digital y segura.

## 🚀 Funcionalidades Principales
* **Gestión de Pacientes:** Registro, edición y consulta de fichas médicas.
* **Sistema de Turnos:** Asignación y seguimiento de citas médicas.
* **Panel Administrativo:** Control total de usuarios (Secretaría y Médicos).
* **Interfaz Dinámica:** Experiencia de usuario optimizada con reactividad.

## 🛠️ Tecnologías Utilizadas
* **Backend:** Python & Django (Django Rest Framework).
* **Frontend:** Vue.js, HTML5, CSS3 (Bootstrap) y JavaScript.
* **Base de Datos:** SQLite (configurada para desarrollo).

## 📦 Instalación y Uso
1. Clonar el repositorio.
2. Crear un entorno virtual: `python -m venv venv`.
3. Instalar dependencias: `pip install -r requirements.txt`.
4. Ejecutar migraciones: `python manage.py migrate`.
5. Iniciar servidor: `python manage.py runserver`.
